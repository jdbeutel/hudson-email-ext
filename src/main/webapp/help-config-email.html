This lets you customize who recieves emails for each build status.

Selecting the checkboxes will cause Hudson to send emails to the checked group, for the corresponding build status.

  • Send To Recipient List - an email will be sent to the recipients that were specified in the Global Recipient List.
  • Send To Committers - an email will be sent to all users who checked in changes since the last build.
  • More Configuration - The contents of the email for each build status can be edited by clicking the (expand) button. You can hide the edit fields again by clicking the (collapse) button. Any changes you make before hiding the fields again will still be saved.
